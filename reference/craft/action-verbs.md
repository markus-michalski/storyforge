# Action Verb List

Used by the snapshot detector in `manuscript_checker.py` to classify sentences
as `action` vs `descriptive`. A sentence containing any of these verbs (or their
common inflections: -s, -ed, -ing) breaks a static description block.

List entries are base verb forms. The detector checks: verb, verb+s, verb+ed,
verb+ing (and drops trailing -e before -ing where needed).
