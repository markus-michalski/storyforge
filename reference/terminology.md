# StoryForge Terminology

## Project Structure

| Term | Definition |
|------|-----------|
| **Book** | A single writing project (short story to epic novel) |
| **Chapter** | A numbered section of a book, stored as a directory with README.md + draft.md |
| **Scene** | A continuous unit of action within a chapter (optional subdivision) |
| **Series** | A collection of connected books sharing world/characters |
| **Draft** | The prose text of a chapter (draft.md) |
| **Outline** | The structural plan for a chapter (in README.md) |

## Author System

| Term | Definition |
|------|-----------|
| **Author Profile** | A writing persona with defined style, voice, and preferences |
| **Studied Work** | A PDF/text analyzed to extract style patterns for an author profile |
| **Voice** | The author's distinctive way of writing (rhythm, vocabulary, tone) |
| **Banned Words** | AI-tell vocabulary that must never appear in prose |
| **Preferred Words** | Character-specific vocabulary that reinforces the author's voice |

## Craft Terms

| Term | Definition |
|------|-----------|
| **GMC** | Goal / Motivation / Conflict — the character framework |
| **Want vs. Need** | External desire vs. internal growth requirement |
| **The Lie** | The false belief a character holds that drives their arc |
| **Fatal Flaw** | A character flaw that actively causes problems |
| **Scene-Sequel** | Dwight Swain's pattern: Goal→Conflict→Disaster / Reaction→Dilemma→Decision |
| **Plant & Payoff** | Foreshadowing technique: introduce element early, pay it off later |
| **Chekhov's Gun** | If you show a gun in Act 1, it must fire by Act 3 |
| **Midpoint Reversal** | A major shift at the story's midpoint that raises stakes |
| **Deep POV** | Writing without filter words, eliminating narrative distance |
| **AI Tells** | Vocabulary and structural patterns that reveal AI-generated text |

## Status Terms

| Term | Used For | Progression |
|------|----------|-------------|
| **Idea** | Book | Initial concept, not yet structured |
| **Concept** | Book | Premise, theme, and conflict defined |
| **Drafting** | Book | Actively writing chapters |
| **Outline** | Chapter | Structure planned, not yet written |
| **Draft** | Chapter | First version of prose complete |
| **Final** | Chapter | Fully revised and approved |
| **Export Ready** | Book | All chapters final, ready for Pandoc |

## Genre Terms

| Term | Definition |
|------|-----------|
| **Base Genre** | A standalone genre (horror, fantasy, romance, etc.) |
| **Cross-Cutting Genre** | A modifier combined with base genres (lgbtq) |
| **Mix Genre** | A pre-defined combination with its own conventions (dark-fantasy) |
| **HEA** | Happily Ever After — mandatory romance ending |
| **HFN** | Happy For Now — alternative romance ending |
| **Hard Magic** | Magic system with clear rules, costs, and limitations |
| **Soft Magic** | Magic system that maintains mystery and wonder |
