# {{title}} — Canon Log (Story Bible)

> **IMPORTANT:** This file is the single source of truth for all established facts in the story.
> The chapter-writer MUST load this file before writing ANY chapter.
> After writing or revising a chapter, update this file with new or changed facts.

## How to Use

1. **Writing a new chapter:** Read the full canon log. Never contradict an established fact.
2. **Revising a chapter:** Update changed facts in the log. Mark old version as `CHANGED`. Flag downstream chapters as `[STALE]`.
3. **Checking continuity:** Compare chapter claims against this log.

## Status Legend

- **ACTIVE** — Current canonical fact, established and unchanged
- **CHANGED** — Fact was revised. Old version noted. Downstream chapters need checking.
- **SUPERSEDED** — Fact was replaced by a later development in the story (not a revision, but a plot development)
- **[STALE]** — Chapter may reference outdated facts due to upstream revision
