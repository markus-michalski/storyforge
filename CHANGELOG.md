## [Unreleased]

### Added
- Initial plugin scaffold with full directory structure
- Plugin manifest (plugin.json, marketplace.json, .mcp.json)
- Configuration system (config.py, paths.py, config.example.yaml)
- State management with StateCache, indexer, and frontmatter parsers
- FastMCP server with 28 tools (state, authors, content, analysis, series, genres, references, quality gates)
- 25 skills covering the full writing pipeline
- 9 templates (book, chapter, character, author-profile, plot-outline, world-setting, synopsis, front/back-matter)
- 18 craft reference documents (73,000+ words) covering story structure, plot, tension, characters, arcs, chapters, dialog, pacing, show-dont-tell, POV, prose style, themes, conflict, openings/endings, world-building, revision, dos-and-donts, anti-AI patterns
- 9 genre-specific craft references (horror, fantasy, thriller, romance, sci-fi, mystery, literary, supernatural, LGBTQ+)
- 14 genre definitions with conventions, tropes, anti-patterns, and mixing notes
- Analysis tools: word counter, style analyzer (AI-tell detection), continuity checker
- PDF/EPUB/DOCX text extraction for author style analysis
- Pandoc wrapper for EPUB/PDF/MOBI export
- Author profile system with style extraction and banned-word lists
- Series support with shared canon and timeline management
- Voice checker with 7-dimension authenticity scoring
- Export reference docs (Pandoc guide, EPUB standards, cover specs)
- Translation reference (best practices for fiction translation)
- Meta references (terminology, model strategy, state schema)
